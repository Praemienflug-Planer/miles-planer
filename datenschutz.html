3. Hosting über GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website können technisch notwendige Verbindungsdaten verarbeitet werden, etwa IP-Adresse, Browserinformationen, Datum und Uhrzeit des Zugriffs sowie abgerufene Inhalte.

Diese Verarbeitung ist technisch erforderlich, damit die Website ausgeliefert werden kann.

4. Nutzung des Prämienflug-Rechners

Wenn du den Prämienflug-Planer nutzt, werden die von dir eingegebenen Werte zur Berechnung verarbeitet. Dazu können unter anderem Ziel, Personenanzahl, Reiseklasse, Reisezeit, Punktestände und Sammelraten gehören.

Diese Eingaben dienen ausschließlich dazu, die gewünschte Berechnung auszuführen. Sie werden nicht für eine individuelle Finanzberatung verwendet.

5. Google Sheets / Google Apps Script

Für einzelne Berechnungen kann eine Google-Sheets- beziehungsweise Google-Apps-Script-Infrastruktur genutzt werden. Dabei werden die eingegebenen Rechnerdaten technisch verarbeitet, um das Ergebnis zu erzeugen.

Bitte gib in den Rechner keine sensiblen personenbezogenen Daten ein. Die Eingaben sollten nur grobe Planungswerte enthalten, zum Beispiel Meilenbestand, Zielregion, Reisezeit und Sammelrate.

6. Cookieless Webanalyse mit Plausible

Zur Verbesserung der Website nutze ich eine datensparsame Webanalyse mit Plausible Analytics. Dabei werden aggregierte Seitenaufrufe und ausgewählte Klickereignisse gemessen, zum Beispiel der Start des Rechners, eine Berechnung, eine Ergebnisanzeige oder Klicks auf weiterführende Sammelwege.

Die Webanalyse erfolgt ohne Tracking-Cookies. Es werden keine Informationen im LocalStorage oder SessionStorage gespeichert, keine User-IDs vergeben, keine geräteübergreifenden Profile erstellt und keine einzelnen Besucher wiedererkannt.

Die Auswertung erfolgt aggregiert. Ich sehe also beispielsweise, wie häufig der Rechner gestartet wurde oder welche Sammelwege angeklickt wurden, aber nicht, welche konkrete Person diese Aktionen ausgeführt hat.

Technisch werden für die Messung unter anderem Seitenpfad, Referrer, Gerätetyp, Browserinformationen und Ereignisnamen verarbeitet. Die IP-Adresse wird nach Anbieterangaben nicht dauerhaft im Klartext gespeichert, sondern nur kurzzeitig zur Erstellung aggregierter Statistiken verarbeitet.

Rechtsgrundlage für diese datensparsame Reichweitenmessung ist mein berechtigtes Interesse gemäß Art. 6 Abs. 1 lit. f DSGVO, die Website, den Rechner und die Nutzerführung zu verbessern. Sofern rechtlich eine Einwilligung erforderlich wird oder ein anderes Tracking eingesetzt wird, wird die Umsetzung entsprechend angepasst.

7. Conversion-Ereignisse

Zusätzlich zu einfachen Seitenaufrufen werden bestimmte Ereignisse erfasst, um die Nutzerführung zu verbessern. Dazu gehören insbesondere:

  • Start des Prämienflug-Rechners
  • Absenden einer Berechnung
  • Anzeige eines Ergebnisses
  • Klicks auf weiterführende Seiten zu PAYBACK, Amex, Miles & More oder Kreditkarten-Einordnungen
  • Klicks auf Kontakt- oder Empfehlungslink-Anfragen

Diese Ereignisse werden nicht mit Namen, E-Mail-Adressen oder anderen direkt identifizierenden Daten verbunden.

8. Kontaktformular

Wenn du das Kontaktformular nutzt, werden die von dir eingegebenen Daten verarbeitet, um deine Anfrage beantworten zu können. Dazu gehören insbesondere Name, E-Mail-Adresse, Betreff und Nachricht.

9. Formspree

Das Kontaktformular kann über den Dienst Formspree verarbeitet werden. Die eingegebenen Daten werden dabei an Formspree übermittelt, damit die Nachricht zugestellt werden kann.

10. Affiliate-Links und Empfehlungen

Klicks auf entsprechende Informations- oder Kontaktseiten können im Rahmen der cookieless Webanalyse aggregiert gezählt werden. Eine personenbezogene Auswertung einzelner Nutzer findet dabei nicht statt.

11. Keine personalisierte Werbung
