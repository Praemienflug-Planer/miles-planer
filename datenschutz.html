Datenschutzerklärung

Informationen zur Verarbeitung personenbezogener Daten auf dieser Website.

Der Schutz deiner persönlichen Daten ist mir wichtig. Diese Datenschutzerklärung informiert darüber, welche Daten auf dieser Website verarbeitet werden.

Diese Website ist bewusst datensparsam aufgebaut. Es werden keine Nutzerkonten, keine Mitgliederbereiche und keine personenbezogenen Nutzerprofile erstellt.

3. Hosting über GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website können technisch notwendige Verbindungsdaten verarbeitet werden, etwa IP-Adresse, Browserinformationen, Datum und Uhrzeit des Zugriffs sowie abgerufene Inhalte.

Diese Verarbeitung ist technisch erforderlich, damit die Website ausgeliefert werden kann.

4. GitHub Traffic-Statistiken

GitHub stellt Repository-Betreibern im Bereich „Insights > Traffic“ einfache, aggregierte Statistiken bereit. Dazu können beispielsweise Seitenaufrufe, eindeutige Besucher, verweisende Websites und beliebte Inhalte gehören.

Diese Auswertung erfolgt durch GitHub im Rahmen des Hostings beziehungsweise der Plattformfunktionen. Auf dieser Website ist kein eigenes externes Analyse-Script wie Google Analytics, Plausible, Matomo oder Meta Pixel eingebunden.

Ich nutze diese GitHub-Statistiken nur als grobe Reichweitenübersicht, nicht zur personenbezogenen Auswertung einzelner Besucher.

5. Nutzung des Prämienflug-Rechners

Wenn du den Prämienflug-Planer nutzt, werden die von dir eingegebenen Werte zur Berechnung verarbeitet. Dazu können unter anderem Ziel, Personenanzahl, Reiseklasse, Reisezeit, Punktestände und Sammelraten gehören.

Diese Eingaben dienen ausschließlich dazu, die gewünschte Berechnung auszuführen. Sie werden nicht für eine individuelle Finanzberatung verwendet.

6. Google Sheets / Google Apps Script

Für einzelne Berechnungen kann eine Google-Sheets- beziehungsweise Google-Apps-Script-Infrastruktur genutzt werden. Dabei werden die eingegebenen Rechnerdaten technisch verarbeitet, um das Ergebnis zu erzeugen.

Bitte gib in den Rechner keine sensiblen personenbezogenen Daten ein. Die Eingaben sollten nur grobe Planungswerte enthalten, zum Beispiel Meilenbestand, Zielregion, Reisezeit und Sammelrate.

7. Keine eigene Webanalyse und kein Tracking

Auf dieser Website werden derzeit keine eigenen Webanalyse- oder Trackingdienste eingesetzt. Insbesondere werden keine Tracking-Cookies, kein LocalStorage-Tracking, keine User-IDs, keine geräteübergreifenden Profile und keine personalisierten Werbeprofile verwendet.

Es werden außerdem keine Meta-Pixel, keine Google-Ads-Remarketing-Tags, keine Session-Recordings und keine Heatmaps eingesetzt.

8. Kontaktformular

Wenn du das Kontaktformular nutzt, werden die von dir eingegebenen Daten verarbeitet, um deine Anfrage beantworten zu können. Dazu gehören insbesondere Name, E-Mail-Adresse, Betreff und Nachricht.

9. Formspree

Das Kontaktformular kann über den Dienst Formspree verarbeitet werden. Die eingegebenen Daten werden dabei an Formspree übermittelt, damit die Nachricht zugestellt werden kann.

10. Affiliate-Links und Empfehlungen

Derzeit werden keine öffentlichen Kreditkarten-Freundschaftswerbungslinks direkt auf der Website eingebunden. Falls künftig Partnerlinks oder Affiliate-Links genutzt werden, werden diese transparent gekennzeichnet.

11. Deine Rechte

Du hast nach Maßgabe der gesetzlichen Vorschriften das Recht auf Auskunft, Berichtigung, Löschung, Einschränkung der Verarbeitung sowie auf Widerspruch gegen die Verarbeitung deiner personenbezogenen Daten.

Wenn die Verarbeitung auf berechtigten Interessen beruht, kannst du der Verarbeitung aus Gründen, die sich aus deiner besonderen Situation ergeben, widersprechen.

12. Änderungen

Diese Datenschutzerklärung kann angepasst werden, wenn sich technische oder rechtliche Anforderungen ändern oder weitere Dienste eingebunden werden.
